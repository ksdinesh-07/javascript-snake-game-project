# 🐍 JavaScript Snake Game

A classic **Snake Game** built using **HTML5 Canvas, CSS, and Vanilla JavaScript**. Control the snake, collect food to increase your score, and avoid colliding with the walls or the snake's own body.

## 🚀 Features

* 🎮 Classic Snake gameplay
* 🍎 Random food generation
* 📈 Real-time score tracking
* 💥 Wall collision detection
* 🐍 Self-collision detection
* ⌨️ Keyboard controls using arrow keys
* 🎨 Built with HTML5 Canvas for smooth rendering

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* HTML5 Canvas API

## 🎮 Controls

| Key            | Action     |
| -------------- | ---------- |
| ⬅️ Left Arrow  | Move Left  |
| ➡️ Right Arrow | Move Right |
| ⬆️ Up Arrow    | Move Up    |
| ⬇️ Down Arrow  | Move Down  |

## ▶️ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/ksdinesh-07/javascript-snake-game-project.git
```

2. Navigate to the project folder:

```bash
cd javascript-snake-game-project
```

3. Open `index.html` in your web browser.

No additional dependencies or installation are required.
